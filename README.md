## Project Overview
As a selected **Student Digital Ambassador**, this repository serves as my official Digital Literacy Portfolio. It documents my work across five core modules designed to help peers navigate the digital world safely, professionally, and efficiently.

## Repository Structure
This repository is organized into five task-based folders as per the project requirements:
* `task-1-presentation/`: Digital Literacy Awareness Infographic.
* `task-2-portfolio/`: Screenshots of professional profiles (GitHub, LinkedIn, Kaggle).
* `task-3-platforms/`: Proof of coding challenges and collaboration tools.
* `task-4-email-etiquette/`: Professional email drafts and social media checklists.
* `task-5-cybercrime/`: Case study on Ransomware and prevention guides.
* `report/`: The final comprehensive Project Report (.pdf/.docx).
